La Trampa Oculta

Mientras exploras el claro, el suelo se desploma bajo tus pies, revelando una trampa oculta con picos afilados. Debes actuar rápido para evitar caer en ella.

    • Opción 1: Saltar hacia atrás para evitar la trampa.
      Opción 2:Tratar de usar alguna herramienta o habilidad para desactivar la trampa.
