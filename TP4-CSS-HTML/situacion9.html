El Pozo Sin Fondo

Caes en un pozo oscuro y profundo, con paredes resbaladizas que dificultan cualquier intento de escape. Debes pensar rápido para encontrar una manera de salir antes de que sea demasiado tarde

    • Opción 1: Intentar escalar las paredes usando fuerza bruta.
      Opción 2:Buscar algún punto débil en las paredes para intentar cavar una salida.
